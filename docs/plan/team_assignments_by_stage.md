## 협업 포인트

**Backend ↔ Frontend**: API 스펙 조율 (각 마일스톤 시작 시)  
**Backend ↔ Data**: 스키마/알고리즘 협의 (M3, M5)  
**DevOps ↔ Backend**: DB 연결, 환경변수  
**QA ↔ 전체**: 테스트 시나리오, 버그 리포트
