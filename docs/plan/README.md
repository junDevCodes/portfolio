# Porest 프로젝트 계획 문서

이 폴더에는 Porest (Portfolio Resume Studio) 프로젝트의 팀 구성, 작업 할당, 개발 체크리스트가 포함되어 있습니다.

## 📚 문서 목록

### 전체 팀 문서
1. **team_composition.md** - 팀 구성 (6명 팀원의 역할, 직급, 성격, 협업 구조)
2. **development_checklist.md** - M0-M5 마일스톤별 개발 체크리스트
3. **team_assignments_by_stage.md** - 마일스톤별 팀원 작업 할당 요약
4. **test_checklist.md** - 종합 테스트 체크리스트 (Unit/Integration/E2E/Performance/Security)

### 팀원별 개별 작업 문서
5. **member_김민준_TL.md** - 김민준 (Team Lead) 개인 작업 문서
6. **member_박지훈_BE.md** - 박지훈 (Backend) 개인 작업 문서
7. **member_이서현_FE.md** - 이서현 (Frontend) 개인 작업 문서
8. **member_최유진_DE.md** - 최유진 (Data Engineer) 개인 작업 문서
9. **member_정하은_DO.md** - 정하은 (DevOps) 개인 작업 문서
10. **member_강민서_QA.md** - 강민서 (QA) 개인 작업 문서

## 🎯 사용 방법

각 팀원은 자신의 개인 작업 문서(`member_xxx.md`)를 참고하여:
1. 담당 개발 단위 확인
2. Task별 체크리스트 수행
3. 완료 시 체크 표시

팀장은 전체 문서를 활용하여 진행 상황을 모니터링하고 조율합니다.
