## 프로젝트 개요

환경 센서 데이터를 분석하여 자연 재해 발생 가능성을 예측하는 머신러닝 모델 개발 프로젝트입니다.

### 주요 기능 및 성과

* 시계열 환경 데이터 분석 및 이상치 탐지 알고리즘 구현
* Random Forest 모델을 활용한 재해 예측 정확도 85% 달성
* 데이터 시각화 대시보드 구축을 통한 직관적인 모니터링 제공
