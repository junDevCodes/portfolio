## 프로젝트 개요

사용자 취향을 분석하여 맞춤형 상품을 추천해주는 개인화 이커머스 플랫폼 'SelF' 개발입니다.

### 주요 기능 및 성과
- Django 기반의 RESTful API 서버 구축 및 DB 모델링
- 협업 필터링(Collaborative Filtering) 추천 알고리즘 구현
- Docker를 활용한 배포 자동화 및 서버 안정성 확보
