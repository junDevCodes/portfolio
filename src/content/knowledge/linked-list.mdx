## 정의
연결 리스트는 각 노드가 데이터와 다음 노드를 가리키는 참조로 구성된 선형 자료구조입니다.
메모리가 연속적이지 않아도 되며, 동적으로 할당될 수 있습니다.

## 특징
- **동적 크기**: 실행 중에 크기가 변할 수 있음
- **삽입/삭제 효율**: 참조만 변경하면 되므로 \(위치를 아는 경우\) \(대개\) O(1)
- **임의 접근 불가**: 인덱스로 접근하려면 순차 탐색이 필요하여 O(n)

## 시간 복잡도(일반적)
| 연산 | 시간 |
|---|---|
| 접근 | O(n) |
| 탐색 | O(n) |
| 삽입 (맨 앞) | O(1) |
| 삭제 (맨 앞) | O(1) |
