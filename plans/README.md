# PoReSt 프로젝트 계획 문서

이 폴더에는 PoReSt (Portfolio Resume Studio) 프로젝트의 개발 계획과 진행 현황 문서가 포함되어 있습니다.

## 문서 목록
1. `plan.md` - 마일스톤 개요와 범위
2. `checklist.md` - 마일스톤별 개발 체크리스트
3. `task.md` - 현재 진행 현황 및 작업 큐
4. `README.md` - 이 문서

## 사용 방법
1. 전체 범위는 `plan.md`를 기준으로 확인
2. 실행 단위는 `checklist.md`에서 체크
3. 진행 현황은 `task.md`에 반영
